## 📌 About This Project

**DataVault** is a single cohesive full-stack web application built task-by-task during the **Cognifyz Technologies Web Development Internship**. Each task introduces new concepts — from basic server setup to advanced features like JWT auth, external APIs, rate limiting, and background jobs.

Each task is **self-contained in its own folder** with full source code, a detailed README, and a LinkedIn video walkthrough.

## 🏗️ Tech Stack

| Layer | Technology |
|---|---|
| Runtime | Node.js v24 |
| Framework | Express.js |
| Templating | EJS |
| Database | MongoDB + Mongoose |
| Authentication | JWT + bcryptjs |
| Styling | Bootstrap 5 + Custom CSS |
| External API | OpenWeatherMap |
| Rate Limiting | express-rate-limit |
| Caching | node-cache |
| Logging | morgan + custom middleware |
| Background Jobs | node-cron |
