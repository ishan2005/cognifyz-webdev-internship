## 🎯 Task Objective

Build advanced interactive UI features: a **real-time password strength meter**, dynamic DOM manipulation, and a client-side **hash-based routing** system for the dashboard — all without page reloads.

## 🧠 Concepts Covered

| Concept | Description |
|---------|-------------|
| **Password Strength Meter** | Live strength calculation + visual bar |
| **Dynamic DOM** | Add/remove/update elements via JS |
| **Hash Routing** | `window.location.hash` for SPA-like navigation |
| **Event Listeners** | `input`, `click`, `hashchange` events |
| **DOM Manipulation** | `querySelector`, `classList`, `innerHTML` |
