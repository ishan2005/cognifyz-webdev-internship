## 🎯 Task Objective

Integrate the **OpenWeatherMap external API** into the dashboard's weather widget. Protect the endpoint with **express-rate-limit** to prevent abuse, and implement API key security best practices.

---

## 🧠 Concepts Covered

| Concept | Description |
|---------|-------------|
| **External API** | Fetch data from OpenWeatherMap REST API |
| **node-fetch / axios** | Server-side HTTP requests |
| **express-rate-limit** | Limit requests per IP per time window |
| **Environment Variables** | Secure API key storage in `.env` |
| **Error Handling** | Graceful failure when API is unavailable |
| **API Proxy Pattern** | Server acts as proxy to external API |
