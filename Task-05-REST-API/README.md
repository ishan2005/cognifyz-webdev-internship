## 🎯 Task Objective

Build a complete **RESTful API** for data entries with all four CRUD operations. The dashboard fetches data dynamically using the **Fetch API** — no page reloads required.

---

## 🧠 Concepts Covered

| Concept | Description |
|---------|-------------|
| **REST Architecture** | Proper HTTP methods and status codes |
| **GET /api/entries** | Fetch all entries for authenticated user |
| **POST /api/entries** | Create a new entry |
| **PUT /api/entries/:id** | Update an existing entry |
| **DELETE /api/entries/:id** | Delete an entry |
| **Fetch API** | Client-side async calls to REST endpoints |
| **JSON Responses** | Structured API response format |
