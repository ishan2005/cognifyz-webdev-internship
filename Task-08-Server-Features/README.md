## 🎯 Task Objective

Add production-grade server features: **morgan HTTP logger**, a **custom structured logger**, **node-cache** for in-memory response caching, and **node-cron** for scheduled background cleanup jobs.

---

## 🧠 Concepts Covered

| Concept | Description |
|---------|-------------|
| **morgan** | HTTP request logging middleware |
| **Custom Logger** | Structured logs (method, URL, status, time) |
| **node-cache** | In-memory caching with TTL |
| **Cache Invalidation** | Clear cache on data mutations |
| **node-cron** | Schedule jobs with cron expressions |
| **Background Jobs** | Midnight cleanup of expired temp data |
| **Performance** | Reduced DB hits via caching layer |
