## 🎯 Task Objective

Integrate **MongoDB Atlas** via Mongoose for persistent data storage. Implement full **JWT-based authentication** with bcrypt password hashing. Protect API routes and EJS pages from unauthorized access.

---

## 🧠 Concepts Covered

| Concept | Description |
|---------|-------------|
| **MongoDB Atlas** | Cloud database setup and connection |
| **Mongoose Schemas** | User and Entry models with validation |
| **bcryptjs** | Password hashing + salt rounds |
| **JWT** | JSON Web Tokens for stateless auth |
| **httpOnly Cookies** | Secure token storage |
| **Protected Routes** | Auth middleware for API + page routes |
| **Pre-save Hooks** | Mongoose middleware for auto-hashing |
